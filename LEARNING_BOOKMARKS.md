**在新服务器部署 OpenClaw Agent**：

完整流程：
1. **环境准备**：Node.js 22+ + pnpm
2. **安装 OpenClaw**：`pnpm install -g openclaw`
3. **恢复工作空间**：从 git 仓库克隆
4. **迁移配置**：`~/.openclaw/openclaw.json`
5. **安全迁移 API 密钥**：使用 `openclaw login` 或安全复制
6. **验证部署**：`openclaw status --all`

关键要点：
- 工作空间应该版本化（AGENTS.md, SOUL.md, TOOLS.md, MEMORY.md）
- API 密钥不要提交到 git，使用 .gitignore
- 配置文件需要手动迁移和调整路径
- 使用 `openclaw login` 重新设置密钥是最安全的方式
