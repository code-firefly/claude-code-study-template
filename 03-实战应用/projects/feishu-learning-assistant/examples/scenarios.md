## 高级场景

### 场景 6：批量操作

如果你需要执行多个操作，可以一次性发送：

```
1. 查看进度
2. 更新 mcp-advanced
3. 创建书签 学习 MCP 服务器开发
```

助手会依次执行每个命令并返回结果。

## 注意事项

1. **网络延迟**：飞书消息到 OpenClaw 可能有 1-3 秒延迟
2. **超时处理**：长时间操作（如初始化缓存）可能超时，建议在本地执行
3. **错误处理**：如果指令未识别，会自动返回帮助文档
4. **书签探索**：书签探索模式下，助手会专注于当前疑问，不会自动返回主线
