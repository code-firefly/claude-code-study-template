## 模块概述

Claude Code 是 Anthropic 官方的 CLI 工具，让你能够通过命令行直接与 Claude AI 协作完成软件开发任务。本模块将系统学习其核心概念、架构、配置和使用方法。

## 核心知识点概览

### 术语速查

| 术语 | 解释 |
|------|------|
| **CLI** | Command Line Interface，命令行界面 |
| **Tool Use** | Claude 调用工具（如编辑文件、运行命令）的能力 |
| **Hook** | 钩子，在特定事件时自动执行的脚本 |

### 1. Claude Code 架构
- CLI 交互模式
- Tool Use 机制
- 权限管理系统
- 配置与钩子系统

### 2. 核心命令
- `/help` - 帮助系统
- `/commit` - 智能提交
- `/edit` - 文件编辑
- `/plan` - 计划模式
- slash 命令系统

### 3. 配置系统
- `CLAUDE.md` 项目级配置
- `.claude/settings.json` 全局配置
- 钩子系统（hooks）
- 权限模式设置

### 4. 安全与权限
- 工具权限控制
- 安全模式
- 命令批准流程
- 风险操作提示

## 常见问题

### Q: 快速模式和完整模式可以切换吗？
**A**: 可以。如果快速模式学完后想深入实践，可以切换到完整模式，补充完成实践部分。

### Q: 需要什么操作系统？
**A**: Claude Code 支持 macOS、Linux 和 Windows（WSL）。建议使用 macOS 或 Linux 以获得最佳体验。

### Q: 遇到错误怎么办？
**A**: 查看官方文档的故障排除部分，或在 exercises/ 目录记录问题并尝试解决。
