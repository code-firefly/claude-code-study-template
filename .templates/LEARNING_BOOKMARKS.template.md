# 学习书签系统

> 用于追踪学习过程中的疑问分支和深入学习点，确保探索后能返回主线。

---

## 使用命令

| 操作 | 命令 |
|------|------|
| 创建书签 | `创建书签 <名称>` |
| 查看书签 | `继续书签` 或 `查看书签` |
| 继续指定书签 | `继续书签 <名称>` |
| 完成书签 | `完成书签 <名称>` |
| 删除书签 | `删除书签 <名称>` |
