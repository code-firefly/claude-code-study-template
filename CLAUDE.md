# AI 技术学习模板 - 工作流程指南

## 项目概述

这是一个 **AI 技术学习模板**，一个可复用的 AI 驱动学习框架。Fork 后即可自定义学习内容，适合系统化学习任何技术领域。

### 学习目标
- 掌握 AI 工具的核心功能和配置
- 学习 MCP (Model Context Protocol) 协议
- 了解 AI 编排和 Agent 配置
- 积累实战项目经验

### 预计学习时长
- **最小可行路径**（P0 模块）：约 1 周
- **核心能力路径**（P0 + P1）：约 6-8 周
- **全栈学习路径**（全部模块）：约 10-14 周

### 模块组织说明
- 每个模块包含：`README.md`、`checklist.md`、`notes.md`
- 知识缓存存储在：`模块名/knowledge/` 目录
- 练习文件存储在：`模块名/exercises/` 目录

## Prompt Skills 说明

本模板使用 **Prompt Skills** 实现按需加载功能模块，减少上下文占用。

| Skill | 功能 | 触发关键词 |
|-------|------|-----------|
| **learning-progress** | 查看状态、更新进度 | "查看学习状态"、"进度怎么样"、"更新进度"、"记录进度" |
| **learning-manager** | 开始学习、完成学习、重置 | "开始学习"、"我要学..."、"学完了"、"搞定"、"重置" |
| **learning-bookmark** | 创建书签、继续探索、完成书签 | "创建书签"、"继续书签"、"我的书签"、"完成书签" |
| **learning-cache** | 初始化缓存、刷新缓存 | "初始化缓存"、"刷新缓存"、"更新缓存"、"拉取资料" |
| **learning-sync** | 检查更新、同步、模块管理、选择性同步 | "检查更新"、"同步学习计划"、"同步核心更新"、"同步课程更新"、"新增模块"、"删除模块" |

### 工作原理

当你说出相关关键词时，对应的 Skill 会自动加载并执行相关功能。这意味着：
- 只加载需要的功能模块，节省上下文
- 仍然支持自然语言表达
- 每个技能职责单一，便于维护

## 技术架构

本模板采用 **渐进式加载** 设计：

```
Level 1: CLAUDE.md (本文件)
         ↓ 仅包含项目概述和快速参考
Level 2: Prompt Skills (按需加载)
         ↓ learning-progress, learning-manager, etc.
Level 3: 详细流程文档
         ↓ 各 Skill 的完整执行步骤
```

这种设计确保：
- 常驻上下文保持精简（~200 行）
- 复杂功能按需加载
- 自然语言触发保持流畅
