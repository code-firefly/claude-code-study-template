## 补充：OpenClaw 简介

OpenClaw 是一个开源的 AI 助手框架（曾用名 Clawbot 和 Moltbot），其特点是：

- **本地优先**: 技能在本地执行，低延迟
- **权限控制**: 技能必须声明文件、网络、命令访问权限
- **版本管理**: 支持技能版本化和回滚
- **社区驱动**: ClawHub 市场拥有 13,729+ 社区技能
- **安全扫描**: 与 VirusTotal 合作扫描所有发布的技能
