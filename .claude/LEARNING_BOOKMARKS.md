## 使用命令

| 操作 | 命令 |
|------|------|
| 创建书签 | `创建书签 <名称>` |
| 查看书签 | `继续书签` 或 `查看书签` |
| 继续指定书签 | `继续书签 <名称>` |
| 完成书签 | `完成书签 <名称>` |
| 删除书签 | `删除书签 <名称>` |
